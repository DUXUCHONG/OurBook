# 学校综述

本文于2013年录入/高2012级13班李璐

中国农业大学，我经过单一的机缘巧合进入的学校，当初因为专业和父母难得地气氛紧张了几天，因为带有农字的名字可能很多人第一排除的就是它，而我一锤定音的契机是211+985。

才来到农大时发觉它的军训很水，那时候对大学还没有什么概念，但是很新奇，直到开学上课才发现我选的这个专业有多么的苦逼，后面专业介绍会详细写，这里主要还是说说学校全方位的情况，衣食住行（农大是分东西两个校区，大一都在东校区，大二有的专业会去西区，璐子的专业就是去西区的==西区各种偏僻，哎，以下都是东区的内容）。

大家的穿着可是非常非常地朴素啊，和农大的校风：团结、朴实、求是、创新，非常吻合，不像很多其他学校，干干净净的小朋友读了几年大学就变成浓妆艳抹的性感之神了（当然凡事从V都有两面性，太朴素了也不好）。

农大的食堂，怎么说呢，当初是被震惊到了，有一个食堂是承包出去的，虽然小贵，但是很华丽；其他几个食堂也都不错，价格适中，味道也还行，周围呢，小吃也特别多，像什么炉肉火烧啊，火锅啊，烤羊肉啊应有尽有，估计跟北京是首都也很有关系，各地美食，虽然不一定正宗，但是味道真的有。

住的地方，估计就和一般的大学差不多吧，四人间，六人间都有，分到什么样的看人品。

农大在北京的四环到五环之间，但是公交还是挺方便的，不得不吐槽北京真的很堵，而且出租车司机拽得跟皇帝一样，来北京这么久就没碰见几个态度好的。还有不得不说的就是农大这个地理位置极极好在哪呢，和清华北大都很近，空闲的话可以长期去蹭课蹭讲座蹭各种表演什么的，还有学院路共同体选课，可以同时听林大，矿大，北语，中国地质等等等等学校的课，这一条路上全是大学，学术氛围非常好。

有关志愿填报的话，分数570-610的同学，考其他好一点的学校选不了好专业又非常想来北京的话，强烈推荐农大，我接触过的人中都觉得农大性价比非常好，很多北京的本地人，考不上清北人师就来农大。地理优势，国家政策优势，眼界优势，真真不一样。并且，特别好的是农大转专业非常容易（当然只是相对于其他一般学校来说），农大最好的两个，生命科学实验班和食品学院，实验班是高考录25个，进校之后有个考试（考生物和英语），再收25个，这跟你的高考分完全没关系多好啊，只要生物学过竞赛英语够硬就随便入了。比如你570分，只能进个不怎么样的专业，但是读了一学期之后可以申请转专业到食院去，通过率大概50%左右。而且这里面水分比较少，就是靠关系的孩子不是特别多，大多都是靠实力（班上30个人，4个同学都是硬转，全转成功了）。
